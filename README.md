# parse_proberserk

1. Клонирование репозитория: git clone https://github.com/EshiNanase/parse_proberserk.git
2. Установить python3.11+
3. Установить зависимости: pip3 install -r requirements.txt
4. Для парсинга турнира обновить TOURNAMENT_URL
5. Для загрузки колод обновить EMAIL, PASSWORD, TOURNAMENT_LINK
6. Для парсинга турнира python3 parse_tournament.py
7. Для загрузки колод python3 upload_deck_lists.py
